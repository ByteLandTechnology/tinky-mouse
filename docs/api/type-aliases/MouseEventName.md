# Type Alias: MouseEventName

> **MouseEventName** = `"left-press"` \| `"left-release"` \| `"right-press"` \| `"right-release"` \| `"middle-press"` \| `"middle-release"` \| `"scroll-up"` \| `"scroll-down"` \| `"scroll-left"` \| `"scroll-right"` \| `"move"`

Represents the type of mouse event.

- `*-press`: Mouse button pressed.
- `*-release`: Mouse button released.
- `scroll-*`: Scroll wheel action.
- `move`: Mouse movement (requires tracking enabled).
