# Function: useMouseContext()

> **useMouseContext**(): [`MouseContextValue`](../interfaces/MouseContextValue.md)

Hook to access the MouseContext.

Must be used within a `MouseProvider`.

## Returns

[`MouseContextValue`](../interfaces/MouseContextValue.md)

The mouse context value containing `subscribe` and `unsubscribe` methods.

## Throws

Error if used outside of a `MouseProvider`.
